# 🏥 Healthcare Cost Exposure Analyzer

An intelligent machine learning application that predicts annual medical costs and simulates lifestyle improvements to discover a **Path to Savings**. 

🚀 **[Live Demo: Check it out on Vercel](https://healthcare-cost-analyzer-frontend.vercel.app/)**

---

## 🛠️ Project Overview
This project was developed for a **Design and Analysis of Algorithms (DAA)** university course. It uses a machine learning model to estimate financial exposure based on 13 key health variables.

### Key Features:
* **Predictive Analysis:** Estimates medical claim costs based on age, BMI, habits, and geography.
* **Financial Risk Mapping:** Calculates what percentage of annual income would be consumed by medical events.
* **AI Path to Savings:** Dynamically suggests lifestyle changes (e.g., activity levels, tobacco cessation) that directly lower projected costs.
* **Aquamorphic UI:** Modern, glassmorphism-inspired frontend built for high readability and mobile responsiveness.

---

## 🏗️ Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (Fetch API).
* **Backend:** Python, FastAPI.
* **Machine Learning:** Scikit-learn, Pandas, Joblib.
* **Hosting:** Vercel (Frontend), Hugging Face Spaces (Backend API).
